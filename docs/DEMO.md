# Spox Demo

This document contains screenshots and demo information for the Spox Flutter Spotify Clone app.

## Screens

### Home Screen
Browse featured playlists, new releases, and personalized recommendations from the Spotify API.

### Search Screen  
Search for tracks, artists, albums, and playlists in real-time using the Spotify Web API.

### Track View Screen
Full-screen now-playing view with album art, lyrics sync, playback controls, sleep timer, and share functionality.

### Stats Screen
Listening statistics including top tracks, top artists, total listening time, and more.

### Equalizer Screen
5-band audio equalizer with presets (Flat, Bass Boost, Treble Boost, Vocal, Electronic) persisted in Hive.

### Offline/Download Screen
Simulated download management for offline listening.

## Running the App

1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your Spotify API credentials
3. Run `flutter pub get`
4. Run `flutter run`
